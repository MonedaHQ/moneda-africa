# Moneda Africa Website

Next.js marketing website for Moneda Africa.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Deployment

This repository now deploys through Vercel's native Git integration.

- Pushes to configured branches are deployed by Vercel.
- GitHub Actions deployment workflows have been removed from this repository.
- Manage environment variables and deployment settings in the Vercel project dashboard.

## TypeScript Migration

The project has been set up for gradual TypeScript adoption:

- TypeScript tooling and type packages are installed.
- A root TypeScript configuration is added.
- JavaScript and TypeScript can coexist during migration.

You can migrate files incrementally from .js/.jsx to .ts/.tsx.
